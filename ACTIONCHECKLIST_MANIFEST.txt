================================================================================
                        PERFORMANCE NOTES
================================================================================

Bundle Size: ~14 KB (uncompressed, gzipped ~4-5 KB)
Re-renders: Only affected steps re-render on prop changes
Memory: Minimal (Set-based expansion state only)
Animations: GPU-accelerated using transform/opacity
Optimal for: 3-10 steps per checklist

Performance Tips:
- Memoize component for large lists
- Use useCallback for action handlers
- Avoid excessive re-renders of parent
- Consider lazy-loading for long workflows
